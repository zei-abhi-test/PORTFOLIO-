# Abhishek BK Portfolio

A premium personal portfolio built with Next.js 15, React 19, TypeScript, TailwindCSS, Framer Motion, GSAP, and Lucide Icons.

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Structure

- `app/` - App Router pages and metadata
- `components/layout/` - Header and footer
- `components/sections/` - Homepage and route sections
- `components/ui/` - Reusable UI primitives
- `lib/site-data.ts` - Portfolio content
